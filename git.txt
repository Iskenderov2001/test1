git init - Инициализирует файл!
git remote add name url - Этот код добавляет удалённый репозиторий на гит хабе!
git pull origin master - Это команда позволяет тянуть все изменение с указанной ветки! (Имя ветки может быть разными! но в нашем случае используется Master!)
git status - Показывает статус изменения файлов! 
git add . - добавляет папку ( поставить точку не забываем! . точка означает текущая директория!)
git commit -m "First commit"
git branch - менеджер веток! показывает список branch
git branch name - создаём новую ветку указывая название ветку
git checkout name branch - Данное команда позволяет менять нашу ветку на другую ветку!
git push origin master - Это команда отвечает за отправку кода на удалённую репозиторий!
git reset filename - с помощью этого кода мы удалаем указанный файл с таким именем!
git push --set-upstream origin <Ветка которая есть на ПК> - Даное команда устанавливает ветку на удалённую репозиторий! и может быть использован только один раз!.
